## How To
### Build
```sh
cd buba_engine
cmake -H. -Bbuild
```

### Test
```sh
cd buba_engine
cmake --build build --target test
```

### Install
```sh
cd buba_engine
cmake --build build --target install
```
Default path are: 
* /usr/local/lib
* /usr/local/include
* /usr/local/pkgconfig
